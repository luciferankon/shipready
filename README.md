# ShipReady

**AI-powered PR descriptions, commit messages & release notes.**

Paste your git diff or describe your changes in plain English. Get polished, professional output ready to ship — instantly.

## Features

- **PR Description** — Title, summary, changes, test plan, and reviewer notes
- **Commit Message** — Conventional Commits format with body and footer
- **Release Notes** — Highlights, what's new, fixes, breaking changes

## How it works

1. Enter your [Anthropic API key](https://console.anthropic.com) (stays in your browser, never sent anywhere else)
2. Paste your git diff or describe your changes
3. Pick a mode: PR Description, Commit Message, or Release Notes
4. Hit Generate (or Cmd+Enter)

## Privacy

Your API key and code never leave your browser. All requests go directly from your browser to Anthropic's API.
